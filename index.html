IDHANY RPG

Idhany is an RPG made by Stev, and is nowhere near done.
It uses Pygame for rendering, and pretty much everything else is made by me.
You can get the game here

Downloading

1. Open the repo in a new tab
2. Click the green "Code" button
3. Click "Download ZIP"
4. Extract the folder (you should be able to right-click the ZIP file and click "Extract")

Modding Guide

1. Download Python from the Microsoft Store
2. run the following command:

pip install pygame

3. Do as you wish!
